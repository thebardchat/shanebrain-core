## Architecture

```
+------------------+
|  Discord User    |
+--------+---------+
         |
+--------v---------+
|  ShaneBrain Bot  |
+--------+---------+
         |
+--------v---------+
|  Load Balancer   |
|  :8000           |
+--------+---------+
         |
    +----+----+
    |         |
+---v---+ +---v---+
| PC A  | | PC B  |
|Primary| |Backup |
+-------+ +-------+
```

**Philosophy:** Local first, cloud backup. Everything runs offline.
